Este proyecto es una aplicación que permite realizar validaciones de código Python.

Entre estas validaciones se incluyen:

- Imports mal ubicados
- Declaraciones de variables
- Comentarios
- Sintaxis de input
- Conteo de operadores

Para ejecutar el proyecto antes debes de tener un archivo con extensión .py para ejecutar.

Instrucciones de uso:

1. Abrir la terminal en la carpeta del compilador .jar
2. Ejecutar el comando: java -jar compilador.jar nombreDelArchivo.py
3. El resultado se mostrará en la terminal
4. Se generará un archivo con el mismo nombre del archivo de entrada con extensión errores.log que contiene el resultado de las validaciones
5. Puedes realizar las validaciones en el archivo .log

Consideraciones:

- El archivo debe tener la extensión .py
- El archivo puede estar en cualquier carpeta
- Verificar la ruta del archivo
- Cada vez que se ejecuta el programa se generará un nuevo archivo errores.log
